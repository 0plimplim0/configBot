# Lattice 
![Static Badge](https://img.shields.io/badge/Status-Pausado-red) ![Static Badge](https://img.shields.io/badge/Lenguaje-Python-blue) ![License](https://img.shields.io)

Bot de discord con un sistema de permisos granular, basado en modulos.

## Características Principales
*   **Modularidad:** Activa solo lo que necesites (Moderación, Logs, Economía, etc.).
*   **Permisos Quirúrgicos:** Define quién puede hacer qué, módulo por módulo.
*   **Arquitectura Python:** Construido con [discord.py](https://github.com) (o la librería que uses).
